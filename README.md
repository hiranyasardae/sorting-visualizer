# Sorting Visualizer

## Description
This project is a Sorting Visualizer developed using HTML, CSS, and JavaScript. It helps students understand how sorting algorithms work by showing the process step by step using visual representation.

## Features
- Allows user to enter custom numbers
- Displays numbers as bars
- Supports Bubble Sort, Selection Sort, and Insertion Sort
- Shows comparisons and swapping visually
- Adjustable speed for better understanding

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open the index.html file in any web browser
2. Enter numbers (example: 5,3,8,1 or 5 3 8 1)
3. Click on a sorting algorithm
4. Observe how the array gets sorted step by step

## Algorithms Used
- Bubble Sort
- Selection Sort
- Insertion Sort

## Conclusion
This project provides a simple and clear way to learn sorting algorithms. It is useful for beginners to understand the logic and working of different sorting techniques.
